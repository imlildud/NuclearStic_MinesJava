# NuclearStic Mines Java Edition

<img src="https://static.wikia.nocookie.net/nuclearstic-mines/images/e/ee/Nuclearlogo.png/revision/latest/scale-to-width-down/180?cb=20251108203724&path-prefix=es" alt="Logo Nuclearstic" width="250"/>

NuclearStic Mines es un juego de estrategia rogue-like busca Minas hecho por Cloud Toons, donde los jugadores en un Tablero generado aleatoria-mente deben cruzar sin tocar las Minas caminando Casilla por Casilla en 4 direcciones cardinales, mostrando el contenido en un área de 5x5 deteniéndose hasta las Casillas que muestran el numero de Minas adyacentes.

El objetivo es llegar hasta la Niña y traerla de regresa por todo el Tablero y pasar al siguiente nivel, al morir pierdes progreso y solo quedara un marcador con tu puntuación.

El Tablero empieza con un tamaño predeterminado de un área cuadrada, la cual generara las Minas aleatoria-mente por las Casillas, al pisar una pierdes automáticamente toda la partida, cada jugador comienza con x Banderines dependiendo del personaje.
